# Rocq Finite Automata

Finite automata in Rocq.

Contains definitions for DFAs and NFAs, along with a proof of equivalence between the two.

Written without LLMs. Sad that this doesn't go without saying :(

Because the equivalence proofs don't require all the properties of FAs (e.g., that there be finitely many states, or that state equality is decidable), there are also some theorems to show that these properties are preserved by the DFA -> NFA and NFA -> DFA conversion functions.
